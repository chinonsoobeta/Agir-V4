# Pilot Observation Scorecard

Use this scorecard after an unsupervised pilot session.

## Quantitative Measures

| Metric                           | Target               | Actual |
| -------------------------------- | -------------------- | ------ |
| time-to-underwriting             | Less than 45 minutes |        |
| time-to-first-report             | Less than 60 minutes |        |
| support intervention count       | 0                    |        |
| unresolved blocker count         | 0                    |        |
| extraction precision issue count | 0 critical           |        |
| trust objection count            | 0 unresolved         |        |

## Qualitative Prompts

- What was the first moment of confusion?
- Which number did you trust least, and why?
- Which step felt slowest?
- Which source document was hardest to verify?
- Did the audit package answer your compliance questions?
- Would you let an analyst repeat this workflow alone?

## Readiness Decision

- Pass: no support intervention, no critical trust objection, and all target artifacts exported.
- Conditional pass: minor workflow confusion but no data integrity issue.
- Fail: fabricated assumption, missing provenance, broken export, or unsupported decision-critical workflow.
